# Item Tracker

Track the quantity and value of gathered or processed items.

## Features

- Track how many of a chosen item you have gathered or produced
- See the current Grand Exchange / live wiki value of tracked items
- Highlight tracked items on the ground and in your inventory
- A side panel summarising tracked quantities and total value
- Optional notifications when thresholds are reached

## Building / running

Run the `run` Gradle task to launch a development client with the plugin loaded:

```
./gradlew run
```
